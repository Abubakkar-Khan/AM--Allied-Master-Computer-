## 🚀 Quick Start

### Prerequisites
- Chrome or Edge Browser (recommended for Web Audio and TTS support).
- A local web server (e.g., `npx serve`, Python `SimpleHTTPServer`).

### Installation
1. Clone the repository.
2. Create a `.env` file in the root directory.
3. Add your [Groq API Key](https://console.groq.com/keys):
   ```env
   GROQ_API_KEY=your_gsk_key_here
   ```
4. Start your local server:
   ```bash
   npx serve . -l 3005
   ```
5. Open `http://localhost:3005` and click **[ ESTABLISH CONNECTION ]**.
